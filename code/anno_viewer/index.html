Welcome to the Sentence Alignment Tool!


This is a short tutorial that will explain the functionality and features of this tool.

This tutorial will cover 4 sections:

  • Header Buttons
  • Making Alignments
  • Sentence Sorting
  • Downloading

Header Buttons

There are a few important buttons located at the top of the tool.

First, there is the upload file button located on the top left. Clicking on this button will allow you to upload your desired alignment file from your computer.

Next, the choose article select bar and the arrow buttons beside the article name is used for navigating to different articles.

Finally, the download button on the top right is used for downloading the present state of annotations and alignments. More detail can be found later in the download section of this tutorial.

Making Alignments

To make an alignment, click on a sentence in the left hand side and then click on the corresponding sentence on the right hand side. Fill out the annotations and click save. Use the shift key to select multiple sentences on either side. The swap button located in the center switches the documents.

Sentence Sorting

This tool supports the unique feature of sentence sorting. Using the similarity scores stored in the provided alignment file, the tool has the ability to sort right-hand sentences from most similar to least similar to the left-hand sentence.

The sorted checkbox indicates whether the tool will sort the sentences or display them in the original order with rankings on the side. By default, the tool will sort sentences.

This feature is useful for annotating or checking if you made the correct annotation.

Downloading

When you are done annotating download the current state of your annotations and alignments by clicking on the download button. An alignment file will be downloaded onto you computer with a filename indicating the current date and time.

If you close the window without downloading, you will lose your annotations and alignments forever. Always remember to download.
